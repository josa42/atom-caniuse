# caniuse package

Quick access to the [Can I Use](caniuse.com) support tables.

![](https://raw.githubusercontent.com/josa42/atom-caniuse/master/screenshot.gif)

## Commands
* `can-i-use:show` Show panel
* `can-i-use:update` Update data
